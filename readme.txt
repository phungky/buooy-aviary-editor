== Description ==

Buooy Aviary Editor allows you to utilize the powerful Aviary Photo Editor to make changes right from the WordPress Admin.

Adobe Aviary come with the following powerful features (https://developers.aviary.com/features):

* __Enhance__ : Hi-Def intelligently sharpens and defines; Illuminate fixes poor lighting issues; and Color Fix performs amazing color correction.
* __Effects__ : A curated collection of over 100 of Aviary’s best photo effects.
* __Frames & Stickers__ : Support and enhance your users’ creativity with stunning graphics, word art, embellishments, frames and more, all made by Aviary’s top-notch design team.
* __Crop, Resize & Rotate__ : Quickly reframe, resize and rotate photo with intuitive tools and preset image cropping options 
* __Adjustments__ : Modify brightness, contrast, saturation, warmth and sharpness with our handy adjust tool.
* __Focus, Splash & Meme__ : Create a meme, selectively add or subtract color and change the focus of images (“tilt-shift”) with our Meme, Splash, and Focus tools.
* __Draw & Text__ : Mark up or doodle on photos before sharing them with intuitive Draw and Text tools. Preset colors help you make your point, fast.
* __Cosmetic Tools__ : Remove red eye, whiten teeth, and magically remove blemishes with our convenient set of cosmetic tools.

For support, chat us up at <http://support.buooy.com/>


== Installation ==

1. Upload the full folder to the '/wp-content/plugins/' directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Do I need an API Key? =

Yes you do! You can get your API Key from <https://console.adobe.io/>. More information is found in the settings page after you installed the plugin.

= Does it work on localhost? =

The short answer is NO.
